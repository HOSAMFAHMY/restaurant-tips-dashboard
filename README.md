# 🍽️ Restaurant Tips Dashboard (Streamlit)

A data visualization project designed to explore customer tipping behavior and uncover patterns in restaurant billing data.

This project is part of my data analysis portfolio.

An interactive dashboard built with Streamlit and Plotly to analyze restaurant tipping behavior using the Tips dataset.  
It explores relationships between total bill, tip amount, and customer attributes such as gender, smoking status, and day of visit.

The dashboard provides interactive filters and visualizations to better understand tipping patterns.

## ✨ Features

- Interactive scatter plot (Total Bill vs Tip)
- Bar chart analysis by gender
- Pie charts for smoker status and day of the week
- Box plot for tip percentage distribution
- Dynamic filters (color, size, row, column)
- Download dataset as CSV
- Fast performance using Streamlit caching

---

## 🧰 Technologies Used

- Python
- Streamlit
- Pandas
- Plotly Express

---

## 📁 Dataset

The project uses the built-in Tips dataset containing:

- total_bill
- tip
- sex
- smoker
- day
- time
- size

## ▶️ How to Run Locally

Clone the repository:
git clone https://github.com/HOSAMFAHMY/restaurant-tips-dashboard.git
cd tips-dashboard

Install dependencies:
pip install -r requirements.txt

Run the app:
streamlit run app.py
